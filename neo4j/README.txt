# Exemples pour Neo4J

## Installation de Neo4J avec Docker

```bash
mkdir -p ./neo4j/data
mkdir -p ./neo4j/conf
mkdir -p ./neo4j/plugins

docker-compose up -d
docker-compose logs -f neo4j
```
